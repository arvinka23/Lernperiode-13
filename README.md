## Lernperiode 13 – Arbeitspakete

Das Ziel dieser Lernperiode ist es,  
30 wichtige und praxisnahe Projekte  
mit der Programmiersprache Rust umzusetzen.  

Der Fokus liegt auf tatsächlich  
brauchbaren Projekten, die reale  
Problemstellungen abbilden.  

Der Einstieg erfolgt mit sehr  
einfachen Aufgaben, um grundlegende  
Konzepte von Rust zu verstehen.  

Mit jedem weiteren Arbeitspaket  
steigt der Schwierigkeitsgrad  
schrittweise an.  

Am 7.1.2026 starten folgende APs:  
AP1: Compression with Rust  
AP2: Compression with Rust  
AP3: Read CSV File  
AP4: Read CSV File  

Diese Arbeitspakete bilden  
die Grundlage für den weiteren Lernfortschritt.
