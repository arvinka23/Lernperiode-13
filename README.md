# Lernperiode 13 – Arbeitspakete

## Ziel der Lernperiode

Das Ziel dieser Lernperiode ist es, **30 wichtige und praxisnahe Projekte** mit der Programmiersprache **Rust** umzusetzen.

Der Fokus liegt auf **realen, anwendungsnahen Problemstellungen**, die typische Aufgaben aus dem Alltag eines Softwareentwicklers widerspiegeln.

Der Einstieg erfolgt mit **einfachen Aufgaben**, um die grundlegenden Konzepte von Rust sauber zu verstehen.  
Der **Schwierigkeitsgrad steigt schrittweise** mit jedem weiteren Arbeitspaket.

---

## Arbeitspakete – Übersicht nach Startdatum

### Start: 09.01.2026

Diese Arbeitspakete bilden die **Grundlage** für den weiteren Lernfortschritt und konzentrieren sich auf einfache Dateiverarbeitung.

#### AP1 – Compression with Rust

**Beschreibung:**  
Einführung in die Datenkompression mit Rust.

**Lernziele:**  
- Grundlagen der Datei- und Datenverarbeitung  
- Arbeiten mit externen Crates  
- Verständnis von Streams und Buffern  
- Erste strukturierte Fehlerbehandlung  

**Ergebnis:**  
Ein einfaches Programm zur Kompression von Dateien.

---

#### AP2 – Compression with Rust (Vertiefung)

**Beschreibung:**  
Erweiterung der Kompressionslogik aus AP1.

**Lernziele:**  
- Wiederverwendung von Code  
- Verbesserte Fehlerbehandlung  
- Saubere Projektstruktur  

**Ergebnis:**  
Eine stabilere und besser strukturierte Kompressionslösung.

---

#### AP3 – Read CSV File

**Beschreibung:**  
Einlesen und Verarbeiten von CSV-Dateien.

**Lernziele:**  
- Arbeiten mit CSV-Daten  
- Parsen von Zeilen und Spalten  
- Abbildung der Daten in Rust-Strukturen  

**Ergebnis:**  
Ein Programm, das CSV-Dateien korrekt einliest und verarbeitet.

---

#### AP4 – Read CSV File (Vertiefung)

**Beschreibung:**  
Vertiefung der CSV-Verarbeitung.

**Lernziele:**  
- Fehlerhafte oder unvollständige Daten erkennen  
- Robustere Datenverarbeitung  
- Saubere Trennung von Logik und Datenmodell  

**Ergebnis:**  
Eine zuverlässige CSV-Verarbeitung für praxisnahe Anwendungsfälle.
